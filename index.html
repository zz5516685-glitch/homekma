생성형 AI(Generative AI) 소개

텍스트·이미지·음성 등을 생성하는 인공지능 기술의 기본 개념과 활용 예시를 간단히 설명합니다.

생성형 AI란?

생성형 AI는 학습한 데이터를 바탕으로 새로운 콘텐츠(텍스트, 이미지, 소리 등)를 만들어내는 모델을 말합니다. 대표적인 예로 대형 언어 모델(LLM), 이미지 생성 모델, 오디오 합성 모델 등이 있습니다.

주요 예시

  • 텍스트 생성: 문서 작성, 요약, 번역, 챗봇
  • 이미지 생성: 프롬프트 기반 그림/디자인 생성
  • 음성·오디오: 음성 합성, 음악 생성
  • 코드 생성: 코드 완성 및 예제 생성

활용과 주의사항

활용: 콘텐츠 제작, 프로토타입, 고객 응대 자동화 등. 주의: 편향, 허위정보, 저작권 문제, 개인정보 보호 등 윤리적·법적 고려가 필요합니다.
